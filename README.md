# AI Powered Sign Language Translator

## Overview
This project was developed during the NIT Warangal internship. The system recognizes sign language gestures and translates them into meaningful output using AI and embedded hardware.

## Problem Statement
Communication between hearing-impaired individuals and others can be difficult. This project aims to reduce this communication gap by recognizing hand gestures and converting them into understandable output.

## Features
- Real-time sign language recognition
- ESP32-CAM image capture
- AI-based gesture classification
- OLED display output
- Embedded implementation

## Hardware Used
- ESP32-CAM
- SH1106 OLED Display
- USB-to-TTL Programmer

## Software Used
- Arduino IDE
- Edge Impulse
- Embedded C++

## Project Files
- Source code
- Project report
- Presentation
- Output images
- Demonstration video

## Results
The system successfully recognizes sign language gestures and displays the corresponding output.
